参考文档的这一部分涵盖了Spring框架中不可或缺的所有技术。

其中最重要的是Spring框架的反转控制(IoC)容器。然后就是spring AOP技术，Spring有其自己的AOP框架，非常容易理解，而且满足了百分之八十的JAVA大型应用程序开发的需求。

Spring也支持集成AspectJ(目前大型java应用程序领域，特性最多，维护支持最多的AOP框架)
